# AICE Basic Study Plan

> **기간**: 2025년 8월 12일 ~ 2025년 10월 12일 (총 2개월)  
> **목표**: AICE Basic 인증 취득 및 AI 기초 개념/실습 능력 강화

---

## 학습 목표
1. AICE Basic 인증 시험 범위 전부 숙지
2. 인공지능 기초 이론(머신러닝, 딥러닝, 데이터 분석 등) 이해
3. 파이썬 기반 AI 실습 프로젝트 수행 능력 확보
4. 주차별 복습 및 모의고사로 실전 감각 강화

---

## 주차별 학습 계획

### **1주차** (8/12 ~ 8/18)
- AICE Basic 시험 개요 및 학습 범위 파악
- Python 문법 복습 (자료형, 제어문, 함수)
- Numpy, Pandas 기초

### **2주차** (8/19 ~ 8/25)
- 데이터 시각화 기초 (Matplotlib, Seaborn)
- AI/ML 개념 정리: 지도학습 vs 비지도학습
- 선형회귀, 로지스틱 회귀 이론 + 실습

### **3주차** (8/26 ~ 9/1)
- 분류(Classification) 알고리즘: KNN, Decision Tree
- 모델 성능 평가 지표 (Accuracy, Precision, Recall, F1-score)
- 실습: 간단한 데이터셋 분류 모델 구현

### **4주차** (9/2 ~ 9/8)
- 딥러닝 기초: 퍼셉트론, 신경망 구조
- TensorFlow / Keras 기본 사용법
- MNIST 데이터셋 분류 실습

---

### **5주차** (9/9 ~ 9/15)
- 비지도 학습: K-means, PCA
- 데이터 전처리 기법 심화
- 실습: 이미지 데이터 전처리 & 군집화

### **6주차** (9/16 ~ 9/22)
- 자연어 처리 기초: 토큰화, 워드 임베딩
- 간단한 텍스트 분류 모델 구현
- 시험 대비 핵심 개념 정리 (1차)

### **7주차** (9/23 ~ 9/29)
- 종합 실습 프로젝트: 데이터 분석 + 모델 학습 + 성능 평가
- 기출문제/모의고사 풀이
- 시험 대비 오답노트 작성

### **8주차** (9/30 ~ 10/6)
- 전 범위 최종 복습
- 실전 모의고사 2회 이상 수행
- 시험 직전 체크리스트 작성
